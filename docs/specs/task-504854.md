# Задача:
Добавить возможность выбрать город при создании/редактировании вакансии/резюме.

## Что нужно:
1. Создать модель City(int id, String name);
2. Создать репозиторий (interface CityRepository) и его реализацию (class MemoryCityRepository) для модели City с методом Collection<City> findAll();
3. Через конструктор репозитория добавить три города (Москва, Магадан, Дедовск);
4. Создать сервис и его реализацию для модели City;
5. Обновить модель Vacancy;
6. Обновить метод обновления вакансии в репозитории MemoryVacancyRepository;
7. Обновить vacancies/create.html и vacancies/one.html;
8. Обновить контроллер VacancyController;
9. По аналогии добавить возможность указания города при создании/редактировании резюме кандидата.

## Контекст:
- В коде HTML с помощью Thymeleaf мы заполняем выпадающий список списком городов. Конечное значение будет соответствовать th:value выбранного option. Значение option это id города, однако, пользователю мы выводим название города через th:text. При редактировании мы указываем одному из элементов атрибут th:selected для отображения текущего указаного города.
