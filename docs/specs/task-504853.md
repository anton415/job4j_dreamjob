# Задача:
Добавить поле видимости вакансии.

## Что нужно:
1. Добавить в файл vacancies/create.html элемент checkbox;
2. Добавить поле boolean visible и методы getVisible()/setVisible() в модель Vacancy;
3. Добавить возможность задания visible в формы создания и редактирования вакансии;
4. Поправить метод обновления в репозитории вакансии;
5. Добавить вывод галок в список всех вакансий;

## Контекст:
- Модель данных содержит поле visible и она связана с формой;
- Для корректной работы геттер и сеттер должны быть НЕ isVisible() и setIsVisible();
- Использовать тег Thymeleaf - th:if. Он принимает переменную типа boolean;
  - Если она true, то элемент становится видимым;
  - Если false, то элемент не отображается на экране.
- Для отображения используемых символов использовать Font Awesome.
