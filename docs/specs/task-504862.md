# Задача:
Реализовать тестирование репозиториев.

## Что нужно:
1. Добавить две зависимости:
    - spring-boot-starter-test
    - h2
2. Добавить профили:
    - test
    - production
3. Подключить плагин Liquibase
4. В папке db создать файл dbchangelog.xml
5. В папке db создать файл liquibase.properties
6. В папке db создать файл liquibase_test.properties
7. Создать файл с настройками подключения к тестовой БД src/test/resources/connection.properties
8. Создать тестовый класс ru.job4j.dreamjob.repository.Sql2oVacancyRepositoryTest в директории src/test/java
9. По аналогии написать тесты длы Sql2oCandidateRepository


## Контекст:
- Файл dbchangelog.xml указывает какие нужно выполнять скрипты и в какой последовательности. Скрипты уже созданы.
- Файл liquibase_test.properties нужен для указания настроек подключения к H2. База будет храниться в файловом хранилище в папке проекта.
